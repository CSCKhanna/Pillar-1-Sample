# Pillar 1: Teaching and Learning

Interactive University of Rhode Island themed presentation for sharing through GitHub Pages.

## Open locally

Open `index.html` in a web browser.

## Keyboard access

- Use arrow keys to move through the presentation.
- Press `?` to open the help dialog.
- Press `Esc` to open or close overview mode.
- Use the on-screen accessibility button for screen-reader instructions.

The presentation includes live slide-status announcements and a full slide transcript for screen readers.
